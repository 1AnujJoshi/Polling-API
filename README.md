# Polling-API
 
 ## Getting Started

### Prerequisites

* npm 
  ```sh
  npm install npm@latest -g 
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/Project-Name.git
   ```
2. Install NPM packages
   ```sh
   npm install dotenv mongoose express
   ```
